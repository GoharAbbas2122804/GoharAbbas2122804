## 🎮 Want to See the Globe in Action?

I've created an interactive 3D globe component that shows global connections! While GitHub doesn't support embedded React components in README files, you can:

1. **View it live**: [Visit my portfolio](https://goharabbas.dev) to see the globe in action
2. **Check the code**: [3D Globe Component Repository](https://github.com/GoharAbbas/3d-globe-portfolio)
3. **Try it yourself**: Clone the repo and run it locally!

```bash
git clone https://github.com/GoharAbbas/3d-globe-portfolio.git
cd 3d-globe-portfolio
npm install
npm run dev
```
